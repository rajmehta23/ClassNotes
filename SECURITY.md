# Security Policy

## Supported Versions

Only the latest release version on the main branch is currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within ClassNotes, please do NOT create a public issue.
Instead, submit a security advisory via GitHub Security Advisories or contact the maintainers directly.

All security vulnerabilities will be promptly investigated and addressed within 48 hours.
